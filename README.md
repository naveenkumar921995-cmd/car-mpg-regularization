🚗 Car MPG Prediction using Ridge & Lasso Regression
📌 Project Overview

This project demonstrates the implementation of:

Linear Regression

Ridge Regression (L2 Regularization)

Lasso Regression (L1 Regularization)

on the Car MPG Dataset to analyze how regularization affects model performance and coefficient shrinkage.

🎯 Objectives

Predict car mileage (mpg)

Handle missing values and categorical encoding

Compare Linear vs Ridge vs Lasso performance

Analyze coefficient shrinkage

Evaluate model using R² and RMSE

🛠 Tech Stack

Python

Pandas

NumPy

Scikit-learn

Statsmodels

Matplotlib

Seaborn

Streamlit (for deployment)

📊 Dataset

Car MPG dataset contains:

Cylinders

Displacement

Horsepower

Weight

Acceleration

Model Year

Origin

MPG (Target)

🔬 Model Comparison
Model	Train R²	Test R²
Linear Regression	✔	✔
Ridge Regression	✔	✔
Lasso Regression	✔	✔

Regularization helps reduce overfitting by shrinking coefficient values.

📉 Key Insights

Ridge reduces magnitude but keeps all features.

Lasso performs feature selection.

Regularization improves generalization performance.
